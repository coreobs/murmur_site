Bringing back community services
that you've missed, for the game you love!

We provide Murmur Servers (Mumble) free of charge with an optional Patreon upgrade, which helps cover the server cost. Any additional money made via Patreon is put into more server costs.

Central, Low Ping Solutions

We currently have boxes located in Australia (Sydney) and Asia (Singapore). We use Vultr to provide this service as they have solid hardware, good routing and are budget focused. Check them out, or use our referral code!

No Physical Limits!

Because we use boxes with High Bandwidth, there are no set limits other than port. Each free server comes with 30 slots and 96kb/s audio quality. Yum!

Reliable, Cost Effective Hosting

The minimum you would pay from another provider for the same side Mumble is $10 AUD. Let's just say we run each box at half that price. But don't worry, you'll actually experience better performance with us over most solutions.

Security and You!

While the joke of selling user data has floated around the community for multiple providers, the only details we store are IPs, basic Geological Data and what you put in your Mumble Certificate!
